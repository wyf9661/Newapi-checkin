🚀 NewAPI 配置生成器

轻松生成多账号签到配置，同站点多账号只需填写一次 URL

📋 导入 JSON 配置

粘贴已有的 JSON 配置，一键导入并按站点自动分组填充表单

💡 使用提示：

  • 站点分组：同一站点的 URL 和 CF Clearance 只需填写一次，站点下可添加多个账号
  • 账号信息：每个账号只需填写 Session、用户ID 和备注名称
  • 保存到本地：将配置保存到浏览器本地存储，下次访问自动加载
  • 明文存储：数据以明文形式存储在浏览器中，请注意安全
